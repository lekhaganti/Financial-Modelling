## Projects

### 1. FMCF — Maya's Quest: Financing Atherium Legends
> Time Value of Money · Project Valuation · Mortgage Amortization · Bond Pricing

A 14-scenario financial modeling project structured as a narrative case study. Each scenario isolates a core concept and builds a working Excel model alongside the theory.

**Concepts covered:**
- Present value and future value (lump sum and annuity)
- NPV analysis with and without inflation adjustment
- Loan amortization and mortgage payment structuring
- Bond pricing under APR and EAR conventions
- Duration, modified duration, and convexity
- Bond price–yield sensitivity analysis (data table)

**Key outputs:** NPV of $1,14,006 on a $220k DLC investment · Bond priced at $1,080.84 (APR) and $1,082.69 (EAR) · Modified duration of 4.46 on a 6-year semi-annual coupon bond

→ [`FMCF/`](./FMCF/)

### 2. FinMod — Firm & Project Valuation: Five Equivalent Methods
> FCFF · FCFE · APV · DDM · Residual Income · WACC · Economic Profit

A rigorous single-model proof that five independent valuation methodologies converge to the same firm value when applied consistently to the same set of cash flows. Built on a 5-year explicit forecast with an infinite horizon terminal value.

**Concepts covered:**
- Free Cash Flow to the Firm (FCFF) and Free Cash Flow to Equity (FCFE)
- Adjusted Present Value (APV) with explicit tax shield valuation
- WACC-based DCF with dynamic leverage-adjusted weights
- Dividend Discount Model (DDM)
- Residual Income / Economic Profit framework
- NOPAT, capital charge, and value-added decomposition

**Key outputs:** Firm value of $2,234.92 confirmed across all five methods · Value added of $1,434.92 over $800 invested capital · Tax shield contribution isolated at $52.76

→ [`FinMod/`](./FinMod/)
